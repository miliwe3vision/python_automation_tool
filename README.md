# Python Automation Toolkit

## 📌 Overview

Python Automation Toolkit is a command-line application developed using Python.

This project demonstrates the Python fundamentals learned in Module 1, including:

- Variables
- Data Types
- Functions
- Loops
- Modules
- File Handling
- JSON
- Error Handling
- Git & GitHub

---

## 🚀 Features

### 📂 File Organizer

Automatically organizes files into folders.

Example:

Images/
Documents/
Videos/
Music/
Archives/
Others/

---

### 📄 JSON Student Manager

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student

Data is stored in **student.json**.

---

### 🔐 Password Generator

- Generate Secure Password
- Generate Multiple Passwords
- Password Strength Checker
